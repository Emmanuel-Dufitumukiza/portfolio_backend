### Installation

1. Clone this repository
   ```sh
   git clone https://github.com/Emmanuel-Dufitumukiza/portfolio_backend.git
   ```
2. Installing Required Modules
   ```sh
   npm install
   ```
2. Run the project
   ```sh
   npm start
   ```
